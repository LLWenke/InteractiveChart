# WK_Kchart

#### 介绍
k线图，实现了柔滑缩放，滑动加载更多。长按选中等交互，包含蜡烛图，分时图，交易量图，深度图，MACD，RSI，KDJ，BOLL图表，也包含MA平均指标。新增指标自定义功能！，新增SAR指标支持！,支持图表模组折叠/并列展示模式!
